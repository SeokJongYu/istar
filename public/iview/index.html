iview

an interactive HTML5 visualizer of protein-ligand complex

Demo

iview supports the following keyrings:

  • Mouse wheel: zoon in/out.
  • Left mouse button: rotate the complex.
  • Right mouse button: translate the complex.
  • Ctrl + left mouse button: rotate the ligand.
  • Ctrl + right mouse button: translate the ligand.
