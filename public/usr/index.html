ultrafast shape recognition for ligand-based virtual screening

The input to usr is onefold:

  • A query ligand.

The output from usr is twofold:

  • log.csv.gz: the usr score, molecular properties, link to substance information and supplier list of the entire database of ligands.
  • ligands.pdbqt.gz: the top 1000 ligands.
