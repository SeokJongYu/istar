ADW

a unified and curated colleciton of clinically approved drugs worldwide for repurposing and virtual screening

Downloads and specifications

The ADW database comprises 3,167 unique drugs that have been approved for clinical use by US (FDA), UK (NHS), EU (EMA), Japanese (NHI), and Canadian (HC) authorities.

Download: adw.tar.gz

Included in the gzipped tarball are:

  • properties.tsv: all properties of the compounds, in tab-delimited format. Columns are:
    • ID: ZINC ID of 8 digits
    • MWT: molecular weight (g/mol)
    • LGP: partition coefficient xlogP
    • ADS: apolar desolvation (kcal/mol)
    • PDS: polar desolvation (kcal/mol)
    • HBD: hydrogen bond donors
    • HBA: hydrogen bond acceptors
    • PSA: topological polar surface area (Å2)
    • CHG: net charge
    • NRB: number of rotatable bonds
    • SMILES: SMILES string
    • VAS: vendors and annotations, delimited by |
    • CAS: CAS registry numbers
    • Name: popular name
  • mol2, sdf, pdbqt: directories of 3D structure files in ready-to-screen formats.
  • svg: directory of vector plots of 2D structures.
  • conformers.sdf: 23,106 conformers.

Construction and statistics

The raw data of ADW were retrieved from 3 catalogs of ZINC:

  • DrugBank-approved of version 2014-03-19
  • FDA-approved drugs (via DSSTOX) of version 2012-07-25
  • the NCGC Pharmaceutical Collection (NPC) of version 2012-03-13

Then a three-step curation was applied:

  • Compounds whose web page returned a HTTP 404 error code were removed, as some compounds deposited in ZINC at an early time were later tagged as depleted.
  • Compounds without a valid CAS registry number were removed, as CAS numbers are normally required to query for commercial availability of the compounds.
  • Compounds with a duplicate canonical SMILES string were removed, as these actually refer to different conformations of the same structure.

These filtering criteria resulted in a healthy, useful and non-redundant set of 3,167 approved drugs worldwide.

Conformer generation

Conformers were generated from SMILES strings to permit structural superimposition and shape recognition. Totally 23,016 low-energy and diverse conformers were generated using RDKit release 2015.09.2, specifically its new algorithm named ETKDG that combines the computationally efficient distance geometry method with experimental torsion-angle preferences obtained from small-molecule crystallographic data.

  • Initially, for each SMILES string, up to 50 low-energy conformers were generated and sorted.
  • Thereafter, a greedy pruning was performed, where the conformer with the lowest energy was retained and from then on only those that are at least RMSD 2Å away from already retained conformations were kept.

These settings ensure the retained conformers are both energy minimized and conformationally diverse. Consequently, an average of 7 conformers per compound were generated and retained, with only one conformer retained for slightly more than 20% compounds and a maximum of 50 conformers retained for some compounds. The number of retained conformers generally increases when the number of rotatable bonds increases (from 0 to 15) because of larger conformational space.

Prospective applications

ADW was constructed for the major objective of drug repositioning via virtual screening. To date there have been succeesful prospective applications of ADW. Through ensemble docking of ADW followed by in vitro cytotoxicity assays on cancer cell lines and in vivo experiments on nude mice, two marketed drugs, fluspirilene and adapalene, have been repurposed as anticancer agents.

USR@istar is a freely available web server for ligand-based virtual screening of ADW. The input is a query molecule, and the output is 1000 geometrically and pharmacophorically similar molecules.
